# PE02_CS451_MinhHuynh

Input
The primary input for this application is gathered through the TextInput component, which allows users to enter specific course codes to indicate their preferences.

Process
The program processes data using React Native’s functional component architecture. A ScrollView is implemented to ensure that the extensive curriculum list remains accessible and fluid across various mobile screen sizes. The processing logic maps course data into visual elements while applying a StyleSheet to establish a clear visual hierarchy. This includes styling major headers with a yellow background and utilizing flexbox properties to manage content alignment and spacing throughout the interface.

Output
The final output is a rendered mobile interface featuring an interactive input field and a categorized overview of degree requirements. The UI utilizes distinct yellow headers for visual separation and a persistent bottom navigation bar to provide a professional, structured user experience for navigating the curriculum.
